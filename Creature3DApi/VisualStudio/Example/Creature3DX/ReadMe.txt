========================================================================
    ACTIVEX 控件 DLL : Creature3DX 项目概况
========================================================================

控件向导为您的 Creature3DX ActiveX 控件 
DLL 创建了此项目，其中包含一个控件。

此主干项目不仅说明了编写 
ActiveX 控件的基础知识，而且是编写特定控件
功能的起点。

此文件包含组成 Creature3DX ActiveX 控件 DLL
的每个文件的内容摘要。

Creature3DX.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主
    项目文件。它包含有关生成该文件的
    Visual C++ 版本的信息，以及有关使用应用程序向导选择的平台、
    配置和项目功能的信息。

Creature3DX.h
    这是 ActiveX 控件 DLL 的主包含文件。
    它包括其他项目特定的包含文件，如 resource.h。

Creature3DX.cpp
    这是包含 DLL 初始化、终止和其他
    簿记的代码的主源文件。

Creature3DX.rc
    这是项目使用的 Microsoft Windows 资源列表。
    可使用 Visual C++ 资源编辑器
    直接编辑此文件。

Creature3DX.def
    此文件包含有关运行 Microsoft Windows 
    所需的 ActiveX 控件 DLL 的信息。

Creature3DX.idl
    此文件包含控件
    类型库的对象描述语言源代码。

/////////////////////////////////////////////////////////////////////////////
CCreature3DXCtrl 控件:

Creature3DXCtrl.h
    此文件包含 CCreature3DXCtrl C++ 类的声明。

Creature3DXCtrl.cpp
    此文件包含 CCreature3DXCtrl C++ 类的实现。

Creature3DXPropPage.h
    此文件包含 CCreature3DXPropPage C++ 类的声明。

Creature3DXPropPage.cpp
    此文件包含 CCreature3DXPropPage C++ 类的实现。

CCreature3DXCtrl.bmp
    此文件包含一个位图，容器将在
    CCreature3DXCtrl 控件出现在工具面板上时使用它来表示该控件。此位图
    位于主资源文件 Creature3DX.rc 中。

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

stdafx.h、stdafx.cpp
    这些文件用于生成名为 Creature3DX.pch
    的预编译头(PCH)文件以及名为 stdafx.obj 的预编译类型(PCT)文件。

resource.h
    这是标准的头文件，它定义了新的资源 ID。
    Visual C++ 资源编辑器读取和更新此文件。
